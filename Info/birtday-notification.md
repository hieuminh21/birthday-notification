
BIRTHDAY SERVICE - SYSTEM PLAN

1. Overview

System to automatically send birthday wishes to employees via email.

2. Tech Stack

- Backend: Java 21, Spring Boot

- Frontend: Thymeleaf + Bootstrap

- Database: PostgreSQL

- Build: Maven

- Excel: EasyExcel

- Email: Spring Mail

- Container: Docker, Docker Compose

3. Main Features

- Employee CRUD

- Import Excel

- Template management

- Auto email sending (8 AM daily)

- Logging

4. Architecture

Controller -> Service -> Repository -> Database
